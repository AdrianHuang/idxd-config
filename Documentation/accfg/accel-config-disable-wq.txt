accel-config-disable-wq(1)
==========================

NAME
----
accel-config-disable-wq - disables an accelerator device work queue

SYNOPSIS
--------
[verse]
'accel-config disable-wq' <device name>/<work queue>

EXAMPLE
-------
accel-config disable-wq dsa0/wq0.0
