dsactl-config-group(1)
======================

NAME
----
dsactl-config-group - configure the individual attribute of group under sysfs

SYNOPSIS
--------
[verse]
'dsactl config-group <group name> [<options>]'

EXAMPLE
-------
dsactl config-group group0.0 --token-reserved=1

OPTIONS
-------
-r::
--tokens-reserved::
	specify the tokens reserved for this group, token-reserved should be
	either 0 or 1.
----
-t::
--tokens-allowed::
	specify tokens allowed by this group, tokens-allowed should be either 0
	or 1.
----
-l::
--use-token-limit::
	specify token limit by this group, use-token-limit should be either 0 or
	1.
----
-a::
--traffic-class-a::
	specify traffic-class-a by this group, traffic-class-a should be larger
	than 0.
----
-b::
--traffic-class-b::
	specify traffic-class-b by this group, traffic-class-b should be larger
	than 0.
