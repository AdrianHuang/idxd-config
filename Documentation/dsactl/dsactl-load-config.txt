dsactl-load-config(1)
=====================

NAME
----
dsactl-load-config - load pre-defined the configuration in json format

SYNOPSIS
--------
[verse]
'dsactl load-config' [<options>]

Load and scan through the pre-defined configuration file in json format,
and set the configured attribute into corresponding component.

EXAMPLE
-------
----
# dsactl load-config /usr/dsactl/config.conf
----

OPTIONS
-------
-l::
--log=::
	to set where to output the config's notification


-c::
--config-file::
	to override the default config file with new config file
